# Study Sankalp

Study Sankalp is a student-focused exam preparation workspace.

It helps students plan study tasks, track syllabus progress, revise weak topics, manage mock tests, organize mistakes, and stay consistent during preparation.

## Current Features

- Professional landing page
- Study Planner
- Built-in syllabus tracker
- Topic progress system
- Browser progress saving
- Dashboard structure
- PYQ Practice section
- Mock Test Tracker section
- Mistake Notebook section
- Analytics section
- AI Tutor coming soon

## Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- LocalStorage
- GitHub
- AWS Amplify

## Project Status

Study Sankalp is currently in beta development.

Current focus:

- Complete landing page content
- Improve dashboard progress data
- Build student-focused tracking tools
- Make the website clean, fast, and mobile-friendly

## Product Direction

Study Sankalp is not a lecture platform or coaching replacement.

It is a preparation management tool for serious students who want to plan, track, revise, and improve with clarity.
